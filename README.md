# 🎓 MERN Learning Management System (LMS)

A full-stack Learning Management System (LMS) developed using the MERN Stack. This application allows students to register, enroll in courses, watch video lessons, complete assignments, attempt quizzes, and track their learning progress through an interactive dashboard.

---

## 🚀 Features

- 🔐 User Authentication (JWT)
- 👤 Student Profile Management
- 📊 Dashboard with Statistics
- 📚 Course Listing
- 🎥 Video Lessons
- 📖 My Courses
- 📝 Assignments
- 🧠 Quiz Module
- ❓ Question Management
- 📈 Progress Tracking
- 📄 Notes Download
- 🏆 Certificate Download
- 📱 Responsive UI

---

## 🛠 Tech Stack

### Frontend
- React.js
- Vite
- React Router DOM
- Axios
- CSS

### Backend
- Node.js
- Express.js

### Database
- MongoDB
- Mongoose

### Authentication
- JWT (JSON Web Token)
- bcryptjs

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/Yashii23/LMS.git
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
